Bufdump's graphical user interface

The bulk of the bufdump graphical user interface is devoted to a scrollable text window. Formatted buffers are displayed in this text widget. Bufdump maintains the formatted output of several buffers at a time in the text window so that you can scroll back and forth to do comparisons between buffers. If more text is available than can be displayed in the text area scrollbars will be automatically added. The text area will also grow if you resize the window.

The top part of the bufdump window contains a menu bar. The menubar provides access to most of the commands that bufdump supports.

  • File provides access to the commands that open data sources, packet definition files, and plugin software components
  • Filter provides access to data filtering tools. Bufdumps data filtering tools allow you to select which buffers it will display and to search for both textual and binary patterns within the buffer.
  • Help provides access to the online documentation you are reading at this time

Below the text display area is a toolbar. At present, the only button on the tool bar is a forward pointing arrow button. Clicking this button acquires the next buffer from the data source that matches the current filter criterion, and displays it in the text widget. If the current filter is complex or corresponds to a rare event, this may take some time.

If the current filter involves a bit pattern filter, once the buffer has been acquired, the text widget will display the region of the buffer that contains the first match to this pattern and highlight the first few words of the match. If no bit filter pattern filter is in place, the software will display the first section of the buffer centered in the text area.
