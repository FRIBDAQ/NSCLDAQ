The Filter menu

The Filter menu provides access to commands that allow you to selectively view data. These commands relate to buffer filters and searches within a buffer:

  • Filter->Filter... establishes a buffer filter to restrict the set of buffers that will be displayed.
  • Filter->Search...establishes a search pattern for data within buffers that have already been displayed. A search for the first instance of this pattern is the executed.
  • File->Search Next searches for the next instance of the established search pattern.
