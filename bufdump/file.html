The File Menu

The File menu contains commands that relate to file operations that are supported by the bufdump program. These operations include:

  • Open... to open a data source from which bufdump will read and format buffers
  • Read Packet Defs... to read a packet definition file to augment bufdumps knowledge of the event packets that may be in event buffers
  • Add plugin... to incorporate a packet formatting plugin to help bufdump format the body of packets it encounterd
  • Exit to exit the program.
