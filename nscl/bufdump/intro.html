Bufdump an Introduction

bufdump is a graphical inspector of buffers from the NSCL data acquisition system.

  • For information about the capabilities of this program, see the overview
  • For a starting point about how to operate the user interface, see the Gui pages
  • For information about data sources, see the data sources page
  • For information about how to extend the set of event packets known by the dumper (and therefore enhance its ability to format events), see the packet definition pages
  • For information about packet formatting plugins, how to write them and how to incorporate them into the software, see the plugins documentation
