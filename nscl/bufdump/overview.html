Overview of bufdump

bufdump is a Tcl/Tk script that understands the structure of data buffers produced by the NSCL Data Acquisition System (NSCLDAQ). In addition to understanding the outer structure of data buffers, bufdump understands how to produce understandable dumps of the internal structure of each buffer. This understanding extends to a knowledge of the known registered packet types, as well as the ability to be taught about new packet types and how to support them.

A summary of the program features includes:

  • Graphical user interface makes using this program easy for inexperienced users.
  • Online and offline data sources allow you to look at data already taken as well as data from a live experiment.
  • Sophisticated buffer filters allow you to specify a subset of buffer types as well as required bit patterns and bit pattern sequences. These filters are easily created and modified via a visual filter creator
  • A knowledge of event packet types as well as the ability to read in packet definition files that extend the set of packets that can be identified.
  • Support for formatting plugins that can produce a more readable form of the internals of event packets. Plugins can be dynamically read in at any time.
  • Sophisticated search capabilities including the ability to search the dump of buffers for specific text, regular expressions or bit patterns and bit sequences.
  • Extensive online help which you are reading now as well as offline man pages and web based documentation at http://docs.nscl.msu.edu
